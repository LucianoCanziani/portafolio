Desarrollé la Pokedex para mejorar mis habilidades en el manejo de grandes cantidades de datos con React Js.

## ¿Qué aprendí en este proyecto?

- Hacer llamadas a APIs con "Axios" y manipular esos datos.
- Filtrado de datos mediante inputs de búsqueda por texto y por opciones.
- Usar "Paginación" cuando hay demasiados elementos para mostrar en una página.
- Implementar música y sonido con "Howler".
- Efectos CSS interesantes como la barra de búsqueda.

## Cómo usarlo:

1. Presioná el botón de inicio para acceder a la pokédex.
2. Navegá y filtrá los Pokémon por tipo y nombre. Seleccioná tu favorito para ver más detalles.
3. Detalle del Pokémon: ahí encontrás toda la información de ese personaje.
